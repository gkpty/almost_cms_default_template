ALMOST CMS - Almost CMS is an open source serverless static page generator powered by AWS S3, AWS DynamoDB, GraphQL, AWS Lambda, React.js, Bootstrap 4

I don't really like wordpress, or PHP for that matter, and for whatever wordpress is good its definitely an overkill for the majority of the projects its being used in. for example, a traditional site that has a gallery and a news section were the owner/editor can write posts. Almost CMS is for these people and the goal is that it will empower these people to do a lot with very little because of :

  • Revolutionary serverless backend
  • Uses AWS amplify react framework
  • S3 object storage
  • graphQL API
  • dynamoDB NoSQL database
  • Other Lambda functions
  • Light weight, fast, flexible
  • Ingenious template system with bootstrap 4 which produces raw html in traditional site structure that can be edited by the user
  • A powerful and flexible CLI

The purpose of almost CMS is to serve those customers that can benefit from having a completely customized static site but want to administer a little bit of content in their website like post an article every now and then, upload a product in their product catalog or add an image to their image gallery. Also to reduce the work from maintaining servers.

Almost CMS isn't really a CMS but its got a serverless admin style app that allows an admin user to upload pictures and write posts to an s3 bucket. As the admin adds content, a series of other lambda functions get triggered to create new pages for each article and update the index page for the articles page and the gallery page. There's also another function that gets triggered to re-write the xml sitemap everytime content is changed.
